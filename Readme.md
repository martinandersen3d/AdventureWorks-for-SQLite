# AdventureWorks-for-SQLite

**This database inludes the following tables:**
- BuildVersion
- Address
- Customer
- CustomerAddress
- ErrorLog
- Product
- ProductCategory
- ProductModel
- SalesOrderDetail
- SalesOrderHeader

**Does not include these tables:**
- ProductDescription
- ProductModelProductDescription
